# RE:r.o.u.t.e
Projekt u sklopu predmeta Umrežene igre na Fakultetu elektrotehnike i računarstva

3D kompetitivna parkour-trkaća igra smještena u urbanoj sredini. U ovoj igri, igrači trče 
po krovovima grada utrkujući se tko je najbolji, najbrži i najspretniji. Tijekom utrke, na 
raspolaganju su vam sredstva kojima možete druge onesposobiti ili usporiti dajući Vam 
prednost u utrci. Također, sve što drugima radite, oni Vam to mogu i vratiti.

Developeri: Karlo Baljak, Filip Borić, Ian Marković
